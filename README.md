# 🎮 Guess the Number Game

A fun and interactive number guessing game built with pure **HTML, CSS, and JavaScript** — no frameworks, no libraries.

## 🕹️ Live Demo
> Open `index.html` directly in your browser — no setup needed!

## 🎯 How to Play
1. A random number between **1 and 100** is generated
2. Enter your guess and click **Check**
3. Get hot/cold hints to find the number
4. You have **10 lives** — don't waste them!
5. Try to win in the fewest attempts possible

## ✨ Features
- 🔴 Hot & Cold hint system (very close / getting warm / quite far)
- ♥️ 10 lives with heart icons
- 📊 Attempts counter + Best score tracker
- 🕵️ Previous guesses shown as pills
- ↺ Play Again without page refresh
- ⌨️ Enter key support
- 🚫 Duplicate guess prevention
- 🌌 Dark galaxy UI with gradient design

## 🛠️ Tech Stack
- HTML5
- CSS3 (Flexbox, Gradients, Transitions)
- Vanilla JavaScript (DOM Manipulation, Game Logic)
